# CRC for node.js

This is a basic port/copy of the JavaScript CRC implementation. The module is compatible with RequireJS.

## Functions

The following functions are exported:

    crc8(String)	#=> Number
    crcArc(String)	#=> Number
    crc16(String)	#=> Number
    fcs16(String)	#=> Number
    crc32(String)	#=> Number
    hex8(Number)	#=> String
    hex16(Number)	#=> String
    hex32(Number)	#=> String

## Installation

    npm install crc
